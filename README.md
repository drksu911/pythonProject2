# Мой первый проект
## Проект по созданию виджета банковских операций клиента. 
## В данном проекте реализованы функции:
1. get_mask_card_number - функция принимает на вход номер карты и возвращает ее маску.
2. get_mask_account - функция принимает на вход номер счета и возвращает его маску.
3. mask_account_card - функция обработки информации о картах и счетах.
4. get_date - функция форматирования даты ДД.ММ.ГГГГ.
5. filter_by_state - функция возвращает словари по указанному значению.
6. sort_by_date - функция сортирует словари по дате.
7. ilter_by_currency - функция фильтрует транзакции по указанной валюте и возвращает итератор.

## Генераторы
1. transaction_descriptions - генератор, который возвращает описание каждой транзакции по очереди.
2. card_number_generator - генератор номеров банковских карт в формате XXXX XXXX XXXX XXXX.
